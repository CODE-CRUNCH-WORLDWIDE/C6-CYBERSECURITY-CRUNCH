# C6-CYBERSECURITY-CRUNCH  
Welcome to the official repository for the **CyberCrunch Workshop**! 🚀  
This workshop is designed to introduce you to the exciting world of cybersecurity and Linux. Over 10 weeks, we’ll explore key concepts, tools, and techniques to help you build foundational knowledge and skills in cybersecurity.
