# Week 1 — Challenges

1. **[Challenge 1 — OverTheWire Bandit, levels 0-5](challenge-01-overthewire-bandit.md)** — the canonical free wargame for exactly this point in a security curriculum. (~3 h)

OverTheWire is free, legal, and designed for learners. Levels 0-5 reinforce the Linux skills from C14 inside a security context: SSH, file inspection, hidden files, permissions, file content tricks. By Bandit level 5 you have used `find` with predicates in anger, which sets up Week 2's network discovery work nicely.

Skip the challenge and you will know the lecture material. Do it and you will *feel* the difference between knowing about Unix and being inside Unix as an adversary thinks about it.
